Cos'è Docker?

Docker è una piattaforma software che ci permette di creare, sviluppare e distribuire applicazioni molto rapidamente.

Docker utilizza degli ambienti isolati di sviluppo chiamati Container. Questi ambienti leggeri e veloci permettono a sviluppatori e amministratori di creare, spedire e eseguire app a basso costo e su qualsiasi scala.

Architettura Docker

Docker è basato su una architettura Client - Server, i componenti principali da conoscere sono tre:

Dockerfile: è un file sul quale vengono scritte le istruzione per costruire l'immagine.

Image: è un template usata per eseguire i container.

Container: sono processi in esecuzione che contengono il progetto.

Installazione

Per usare docker bisogna installare il software Docker Desktop, esso permette di vedere tramite gui la lista dei container in esecuzione.

Immagini Docker

Le immagini create possono essere pushate su cloud in registri privati e pubblici

Qualsiasi developer può scaricare un'immagine e ricreare l'ambiente che desidera.

In altre parole un'immagine può essere usata per creare container infinite volte e in infiniti luoghi.

Dockerfile

FROM node:12-alpine
RUN apk add --no-cache python g++ make
WORKDIR /app
COPY . .
RUN yarn install --production
CMD ["node", "src/index.js"]

Dockerfile

Il dockerfile è un file di testo che contiene varie info fondamentali per la creazione dell'immagine.

Nel dockerfile vanno specificate vari dati, come il linguaggio di programmazione che vogliamo usare per l'immagine e qual è la directory del nostro progetto.

Questo file verrà tradotto da Docker Engine e verrà trasformato nell'immagine che necessitiamo.
